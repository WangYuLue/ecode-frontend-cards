### 1、CSS中 px、em、rem、vw、vh、vmin、vmax 这些单位有什么区别？

**px**
pixel的缩写，意为像素

**em**
参考物是父元素的 `font-size`

**rem**
参考物是根元素html的 `font-size`

**vw**、**vh**
1vw等于视窗宽度的1%；
1vh等于视窗高度的1%；

**vmin**
vw和vh中较小的那个

**vmax**
vw和vh中较大的那个

### 3、flex设置成 1 和 auto 有什么区别？

如下样式是等同的：

```css
.item {flex: 1;}
.item {
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: 0%;
}
```

如下样式是等同的：

```css
.item {flex: auto;}
.item {
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: auto;
}
```
